## Description
- This project is a social network API that allows users to share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This application uses Express.js for routing, a MongoDB database, and the Mongoose ODM. The API routes are tested in Insomnia Core.
    
## Installation
- Git clone the repository to your local machine.
- Run `npm install` to install dependencies.
- Run `npm start` to start the server.
- Open Insomnia Core to test the API routes.
